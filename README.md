
# Getting Started with Framework

## Deployment

To deploy this project run

```bash
  git clone https://github.com/kahmad2003/Hybrid-Framework.git
  ```

Make sure to change directory before executing command
```bash
  cd Hybrid-Framework
  ```

Make sure maven is configured
```bash
  mvn clean test
```
